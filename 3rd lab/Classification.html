  3. H desbalanceamento das classes (isto , uma classe tem muito mais instncias que outra)? Em que proporo? Quais efeitos colaterais o desbalancemanento de classes pode causar no classificador?

    4. Sim, h muito mais instncias FALSE do que TRUE para a varivel de resposta, ou seja, h muito mais dados de alunos que no evadiram do que de alunos que evadiram. A proporo pode ser vista na visualizao acima: FALSE - 96,1% e TRUE - 3,9%.

    Com classes desbalanceadas, o classificador pode se tornar enviesado para a classe majoritria. Os algoritmos de classificao tentam diminuir o erro geral da previso, porm com classes desbalanceadas o erro real pode ser enconberto. Os algoritmos em geral assumem que as classes estaro balanceadas e tambm assumem que erros obtidos por diferentes classes possuem o mesmo custo.

    quantas.finais <- function(medias) {
  temp <- medias[medias < 7]
  return(length(temp))
}

dados <- dados %>%
  group_by(matricula) %>%
  mutate(num_finais = quantas.finais(medias = media))

evadiu.true <- dados %>%
  ungroup() %>%
  filter(evadiu == TRUE) %>%
  select(matricula, num_finais) %>%
  group_by(matricula, num_finais) %>%
  unique() %>%
  ungroup() %>%
  select(num_finais) %>%
  table()

evadiu.false <- dados %>%
  ungroup() %>%
  filter(evadiu == FALSE) %>%
  select(matricula, num_finais) %>%
  group_by(matricula, num_finais) %>%
  unique() %>%
  ungroup() %>%
  select(num_finais) %>%
  table()

evadiu.true <- as.data.frame(evadiu.true)

    Cada aluno pode ter feito entre 0 e 6 provas finais. Podemos ver que dentre os alunos que evadiram (evadiu == TRUE), bem mais comum alunos que fizeram quatro ou mais exames finais (73,6% dos alunos) do que alunos que fizeram tres ou menos finais (26,4%). E o contrrio ocorre para alunos que no evadiram. Dentre estes alunos bem mais comum alunos que fizeram 3 ou menos finais (72,1%) do que alunos que fizeram 4 ou mais finais (27,9%). Ento podemos considerar esse um bom atributo pois ele possui distribuies quase que contrrias para as diferentes classes da varivel de reposta. Atributos que possuem distribuies diferentes entre as classes so melhores do que atributos que possuem distribuies parecidas pois eles discriminam melhor a varivel de resposta. Para este caso, temos a intuio de que alunos que fizeram mais finais tm maiores chances de evadir.
